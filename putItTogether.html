Put it Together

A web-based app based on the PokéAPI.

The purpose of this guide is to give you a step-by-step tutorial on how to create a dynamic Pokédex using the PokéAPI. This tutorial is organized into the following sections:

  • Getting Started - An introduction to the PokéAPI.
  • Making an API Request - Learn how to use AJAX to call PokéAPI methods.
  • JSON Response - Learn how to read and parse GET responses in JSON format.
  • Putting It All Together - Use HTML, CSS and JS to put together your own Pokédex.
  • Demo: Dynamic Pokedex - A sample Pokédex using the code in this tutorial.
